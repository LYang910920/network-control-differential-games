# Documents

This directory contains the tutorial documents used by the examples.

```text
lecture_note.pdf
lecture_note.tex
code_walkthrough_and_model_adaptation_guide.pdf
code_walkthrough_and_model_adaptation_guide.tex
```

## Reading order

1. `lecture_note.pdf`
   - Mathematical background for optimal control, differential games, hybrid control, and network models.

2. `code_walkthrough_and_model_adaptation_guide.pdf`
   - Practical guide for running the code, reading the scripts, modifying the model equations, updating Jacobians, changing payoff functions, and validating strategies.
