# Vendored Reference Source Code

This directory contains source-code snapshots from the three upstream reference repositories discussed in the tutorial:

```text
OpinionMalware_TIFS_2025_Code/
PropagandaWar_TIFS_2024_Code/
Propaganda_TCSS_2025_Code/
```
